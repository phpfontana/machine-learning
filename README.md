# Machine Learning 
This repository contains a tutorial on machine learning using popular python libraries. It covers the basics of machine learning and how to train and evaluate machine learning models.

### Prerequisites

To follow along with this tutorial, you will need the following:

- Basic knowledge of machine learning concepts and Python programming
- A computer with a recent version of the libraries installed (check requirements.txt)


## Table of Contents
The tutorials in this repository cover a wide range of topics, including:

**1. Basic Models**
- Linear Regression
- Logistic Regression

**2. Advanced Models**
- Artificial Neural Network
- Decision Tree
- Random Forest
- XGBoost

**4. Unsupervised Learning**
- Clustering
- Anomaly Detection

**5. Recommender Systems**

**6. Reinforcement Learning**
